# password_generator.py

This is my third project in Oasis: a Python Password Generator. The program asks the user to enter their name, phone model, and the desired password length. Based on the given length, the program generates a random and secure password using a combination of uppercase letters, lowercase letters, numbers, and special symbols. This project helped me practice important Python concepts such as loops, strings, user input handling, and the use of built-in modules like the random and string modules.

The program works by first collecting user details such as name, phone model, and the required password length. After receiving the input, it creates a pool of characters that includes letters, digits, and special symbols. Using the random module, the program selects characters randomly from this pool and combines them to generate a strong password of the specified length. The generated password is then displayed to the user in the terminal.

I developed this project using basic Python programming concepts such as variables, loops, and user input. I used the string module to access letters and numbers and the random module to select characters randomly so that the password becomes more secure. While developing this project, I focused on writing simple and clear code so that beginners can easily understand how password generation works in Python. This project helped me improve my logical thinking and understand how Python can be used to create useful security-related tools.
